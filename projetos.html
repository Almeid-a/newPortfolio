PROJETO HBO

Este projeto foi desenvolvido utilizando HTML, CSS e JavaScript.


Trata-se de um site dedicado às séries de sucesso da HBO, fornecendo uma breve descrição de cada uma. Os fãs de séries encontrarão neste projeto uma maneira fácil e organizada de descobrir informações sobre seus programas favoritos, ajudando a tornar a experiência de assistir ainda mais envolvente e informativa.

PROJETO HAMBURGUERIA

Este projeto foi desenvolvido utilizando HTML, CSS com a biblioteca Tailwind e JavaScript.


Trata-se de um site para uma hamburgueria, projetado para simplificar o processo de pedido. Os clientes podem fazer seus pedidos de forma rápida e conveniente, e os pedidos são automaticamente encaminhados para o WhatsApp da hamburgueria, facilitando a comunicação e agilizando o atendimento.

APP - PORFÓLIO

Este é o aplicativo do meu portfólio, desenvolvido usando React Native e estilizado com NativeWind.


O app possui três menus: Home, Projetos e Certificados. A tela inicial serve como uma apresentação, enquanto a segunda mostra os projetos realizados e, por último, a página de formação e certificados exibe minha trajetória educacional e as conquistas obtidas.

APP - CRONOMETRO

Este aplicativo foi desenvolvido utilizando React Native e gerenciamento de estado com AsyncStorage. Para a navegação entre telas, foi utilizado o Navigation Container.


Trata-se de um aplicativo de cronômetro que oferece a funcionalidade de salvar cada intervalo de tempo registrado em um histórico. Isso permite aos usuários acompanhar e revisitar seus registros anteriores, fornecendo uma visão detalhada do tempo decorrido em diferentes momentos.

APP - BISCOITO DA SORTE

Este aplicativo foi desenvolvido utilizando React Native e as propriedades de Component State.


Ele simula um biscoito da sorte, exibindo mensagens aleatórias para o usuário. É uma aplicação leve e divertida para dispositivos móveis.
